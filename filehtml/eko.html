Sistem Operasi

Mengapa Kita Perlu Mempelajari Sistem Operasi?

Sistem operasi merupakan salah satu perangkat lunak komputer, terlebih pada jaringan yang nantinya komputer tersebut digunakan untuk mengkoneksikan ke jaringan internet. Ada banyak alasan dan tujuan yang bisa kita jabarkan mengenai megnapa kita sebagai user wajib mempelajari sistem operasi komputer, meskipun hanya dasarnya. Berikut ini adalah beberapa tujuan mempelajari sistem operasi komputer :



1. Agar user bisa mengoperasikan fungsi utama dari komputer

Tentu saja, seorang user biasa wajib mengetahui fungsi utama dan juga fungsi dasar dari sebuah sistem informasi. Disini user juga dituntut hal paling mendasar bagimana cara menggunakan komputer sesuai dengan prosedur yang benar. Paling tidak, seorang user harus memahami bagiamana cara untuk melakukan beberapa fungsi sistem operasi, seperti:

• Membuat dokumen dan pengetikan

• Melakukan koneksi ke internet

• Membuka file explorer dan juga melihat menu

• Melakukan seleksi dan juga pemilihan sebuah menu

Apabila seorang user tidak mau mempelajari sebuah sistem operasi, maka fungsi-fungsi dasar tersebut tentu saja tidak akan bisa dilakukan oleh user.


2. Agar user memahami fitur yang dari sistem operasi komputer

Selain fungsi dasar yang dimiliki oleh sebuah sistem operasi, mempelajari sistem operasi juga akan membantu user dalam memahami fitur-fitur dasar dan juga kemampuan yang dimiliki oleh sebuah sistem operasi. Hal ini akan membantu user dalam mempermudah pekerjaannya dengan menggunakan sistem operasi komputer. Tentunya fitur yang dimiliki berbeda-beda sesuai dengan jenis sistem operasi komputer yang digunakan.


3. Agar user bisa menyelesaikan pekerjaannya dengan cepat

Dengan mempelajari sistem operasi komputer, maka user akan memahami selah-selah dalam mengoperasikan komputer miliknya. Ketika user sudah memahami hal ini, maka tentu saja user akan menjadi lebih cepat dalam menyelesaikan pekerjaannya dengan baik.


4. Agar user bisa mengoptimalkan perangkat keras yang dimilkinya dengan baik

Optimalisasi perangkat keras, seperti harddisk (baca juga: fungsi harddisk), processor, dan lainnya juga bisa dilakukan dengan sistem operasi. Dengan mempelajari sistem operasi, maka user akan memahami bagaimana cara agar RAM dan harddisk tidak cepat rusak, yaitu dengan membatasi program dan aplikasi yang dijalankan.


5. Untuk kepentingan troubleshooting sederhana

Troubleshooting sederhana, seperti virus ringan dan juga komputer sering hang pun akan bisa menjadi poin penting bagi seseorang yang mempelajari sistem operasi pada sebuah komputer. Apabila seseorang mempelajari sistem operasi, maka dia akan memahami paling tidak cara sederhana untuk menghapus virus, dan juga mengahadapi komputer yang mengalami hang.


6. Agar perawatan sistem operasi, hardware dan software berjalan dengan baik

Tujuan lainnya bagi seseorang dalam mempelajari sistem operasi komputer, terutam apada user adalah agar user memahami bagaimana cara merawat sistem operasi komputer dengan baik dan juga benar. Hal ini akan membantu meningkatkan kelangsungan hidup pada sistem operasi dan juga komponen-komponen komputer menjadi lebih tahan lama. Sepeti contoh kecilnya, pengaruh RAM terhadap kerja komputer, RAM yang merupakan hardware komputer menjadi otak utama dalam komponen komputer.


7. Supaya user memahami mengenai cara mengoperasikan sisem komputer

Yang paling penting, tentu saja adalah seorang user akan mampu untuk memahami bagaimana cara untuk mengoperasikan sebuah komputer dengan menggunakan sistem operasi. Bayangkan apabila seseorang tidak mempelajari sistem operasi terlebih dahulu, maka pasti dia tidak akan bisa mengoperasikan komputer tersebut, meskipun spesifikasi komputernya canggih sekalipun.



Tujuan Mempelajari Sistem Operasi Bagi Superuser

Ketujuh dan beberapa contoh tujuan mempelajari sistem komputer yang sudah dijelaskan diatas merupakan tujuan bagi para user. Akan tetapi tidak hanya user, melainkan superuser atau mereka yang memahami ilmu komputer juga perlu mempelajari mengenai sistem operasi komputer. Mengapa? Berikut ini adalah tujuan dari superuser dalam mempelajari sebuah sistem operasi dari komputer:

• Agar bisa mengoptimalkan kinerja dari sebuah sistem operasi komputer

• Membantu superuser dalam memecehakan masalah sistem operasi yang lebih berat

• Mengembangkan fungsi-fungsi penting dan juga fungsi tambahan dari sebuah sistem operasi komputer

Tentu saja bagi para superuser, kebutuhan untuk mempelajari sistem operasi komputer sangatlah tinggi. Hal ini disebabkan karena kebutuhan dari seorang superuser komputer adalah tidak hanya menggunakan sistem operasi komputer tersebut, namun juga berperan dalam melakukan pengembangan dan juga perawatan atau maintenance dari sebuah sistem operasi yang digunakan dalam komputer. Karena itu, biasanya superuser sanggup melakukan modifikasi dan juga perbaikan dari sebuah sistem operasi, yang mungkin sulit untuk dilakukan oleh user biasa.



Komponen Utama Sistem Operasi

Kernel

Kernel merupakan suatu software (kumpulan program) yang membentuk sistem dan memiliki tugas melayani bermacam program aplikasi untuk mengakses hardware komputer secara aman dan terkendali. Karena akses terhadap hardware terbatas, sedangkan ada lebih dari satu program yang harus dilayani dalam waktu yang bersamaan, maka kernel juga bertugas untuk mengatur agar kapan dan berapa lama suatu program dapat menggunakan satu bagian hardware tersebut. Hal tersebut dinamakan sebagai ”multiplexing”.

Akses kepada hardware secara langsung merupakan masalah yang kompleks, oleh karena itu kernel biasanya mengimplementasikan sekumpulan abstraksi (gambaran umum tentang) hardware. Abstraksi-abstraksi tersebut merupakan sebuah cara untuk menyembunyikan kerumitan, dan memungkinkan akses kepada hardware menjadi mudah dan seragam. Sehingga abstraksi (gambaran umum) ini pada akhirnya memudahkan pekerjaan programmer.

Untuk menjalankan sebuah komputer kita tidak harus menggunakan kernel sistem operasi. Sebuah program dapat saja langsung diload dan dijalankan diatas mesin komputer, yakni bila programmer ingin melakukan pekerjaannya tanpa bantuan abstraksi hardware atau bantuan sistem operasi. Tehnik ini digunakan pada komputer generasi awal, sehingga bila kita ingin berpindah dari satu program ke program lain, kita harus mereset dan me-load kembali program-program tersebut.

Ada 4 kategori :

• Monolitich kernel. Kernel yang menyediakan abstraksi hardware yang kaya dan powerful.

• Microkernel. Kernel yang menyediakan hanya sekumpulan kecil abstraksi hardware sederhana, dan menggunakan aplikasi-aplikasi yang disebut sebagai server untuk menyediakan fungsi-fungsi lainnya.

• Hybrid (modifikasi dari microkernel). Kernel yang mirip microkernel, tetapi ia juga memasukkan beberapa kode tambahan di kernel agar ia menjadi lebih cepat

• Exokernel. Kernel yang tidak menyediakan sama sekali abstraksi hardware, tapi ia menyediakan sekumpulan library yang menyediakan fungsi-fungsi akses ke hardware secara langsung atau hampir-hampir langsung.  

File

File disini merupakan file-file yang dibentuk atau dijalankan oleh sistem operasi. File disini juga berarti sistem berkas dan file system yang dimiliki oleh suatu sistem operasi.

User Interface (Shell)

Sebuah sistem operasi memiliki karakteristik (bentuk) interface (tampilan) yang menjadi interaksi antar user dengan komputer. Bentuk umum user interface yang ada, adalah :

1. Command Line Interface atau command line interpreter (CLI) CLI memberikan tampilan dalam mode teks ke user, dengan backgroud satu warna dan tampilan teks yang juga satu atau beberapa warna dasar.

2. Graphical User Interface (GUI). GUI memberikan tampilan yang lebih interaktif dan nyaman digunakan oleh user. Resolusi gambar dan jumlah warna yang dihasilkan juga lebih banyak, tergantung kemampuan Video Adapter yang dimiliki komputer.
